How it works

Three lines of code.
Complete visibility.

Instrument your agents once. Kairos handles capture, storage, replay, and compliance.

Instrument once

Add 3 lines of code. Every tool call, retry, decision, and output is captured automatically. Works with any agent framework.

from kairos_trace import trace with trace("agent.review") as t: result = agent.run(input)

Replay everything

Step through any execution from start to finish. See exactly where context was lost, which tool failed, why the agent returned that answer.

# Jump to any run kairos replay --run 4417 kairos diff --event context.loss kairos share --link

Export the evidence

Full audit trail for every run. EU AI Act Article 12 compliance documentation generated automatically. Regulator-ready, always.

# Export audit trail kairos export --run 4417 \ --format pdf \ --standard eu-ai-act
