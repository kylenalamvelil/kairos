How it works

Three steps. Complete visibility.

Instrument once. Kairos handles capture, replay, and compliance from there.

01   Instrument

One import. Every decision captured.

Works with any agent framework. Zero configuration. Every tool call, context window, and retry logged automatically.

from kairos_trace import trace with trace("contract_review") as t: result = agent.run(document) t.tag("outcome", result.status)
02   Replay

Scrub any execution. See what your agent saw.

Navigate any past run step by step. Compare runs side by side. Share a link to a specific failure with your team.

# CLI replay kairos replay --run 4417 # Filter by event type kairos diff --event context.loss # Shareable link kairos share --run 4417
03   Prove

Export EU AI Act Article 12 audit trails instantly.

Regulator-ready, always. Export a complete structured audit trail for any execution in seconds.

# Export audit trail kairos export \ --run 4417 \ --format pdf \ --standard eu-ai-act-art12
